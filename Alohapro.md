## 4. 杆件静态形变计算（简化梁模型）

## 4.1 计算模型

采用两级悬臂梁简化：

- 上臂梁：`L2 = 0.36 m`，`I2 = 1.65e-8 m^4`
- 前臂梁：`L3 = 0.37 m`，`I3 = 2.95e-9 m^4`

说明：`I2/I3` 取当前方案等效截面惯性矩口径，用于理论比较，不代表最终 CAD 精确值。

## 4.2 上臂引起的 TCP 形变

上臂端点（J3处）挠度：

`delta_J3 = sum(F_i*a_i^2*(3L2-a_i)/(6EI2)) + F_dist*L2^3/(3EI2)`

上臂端点转角：

`theta_J3 = sum(F_i*a_i*(2L2-a_i)/(2EI2)) + F_dist*L2^2/(2EI2)`

其中：

- 上臂局部载荷：大臂300段、大臂60段、J3电机
- 远端等效载荷：`F_dist = (前臂结构 + J4/J5/J6/夹爪)*g = 14.4818 N`

计算得到：

- `delta_J3 = 0.0003199 m`
- `theta_J3 = 0.0014328 rad`

映射到 TCP：

`delta_tcp_from_J2 = delta_J3 + theta_J3*L3 = 0.0008500 m = 0.8500 mm`

## 4.3 前臂自身引起的 TCP 形变

前臂各点载荷对自由端挠度叠加：

`delta_tcp_from_J3 = sum(F_i*a_i^2*(3L3-a_i)/(6EI3))`

其中 `a_i` 取各质量块到 J3 的距离（0.095, 0.19, 0.245, 0.335, 0.37 等）。

计算得到：

`delta_tcp_from_J3 = 0.0005315 m = 0.5315 mm`

## 4.4 结构总静挠度

`delta_struct = delta_tcp_from_J2 + delta_tcp_from_J3`

`delta_struct = 0.0013815 m = 1.3815 mm`

---

## 5. 与控制精度合成

沿用前文控制链量化结果（含减速比）：

`e_ctrl = 0.388247 mm`

总理论误差（RSS）：

`e_total = sqrt(delta_struct^2 + e_ctrl^2)`

`e_total = sqrt(1.3815^2 + 0.3882^2) = 1.4351 mm`

---

## 6. 结论（本工况）

在“无末端抓取载荷、计入电机重量”下：

1. 关节静态重力矩：`J2 = 12.21 N.m`，`J3 = 3.26 N.m`。  
2. 结构静挠度主导：`~1.38 mm`，明显大于控制量化项 `~0.39 mm`。  
3. 合成理论误差约 `1.44 mm`，理论上**不能达到 0.5 mm**。
