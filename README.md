# 3kg 双臂分拣机械臂（单臂）详细优化说明

> 更新日期：2026-04-09  
> 本版新增：6DoF 对应 5 连杆细分、L3/L4 中置偏转电机方案、日字截面抗形变评估、0.5mm 精度提升路线、简化机械减速设计、断电阻尼力矩整理。

## 1. 目标与本版结论

- 目标负载：末端 `3 kg`
- 目标结构：`6DoF + 1 夹爪`，单臂先设计，双臂镜像
- 关键变化：`L2=350 mm` 固定，L3/L4 细分并加入中置偏转电机（J4 远置驱动）
- 当前理论结果：扭矩满足 3kg，重复定位在工程上约 `2-3 mm`；若要稳定到 `0.5 mm`，必须优先升级 J2/J3 的传动与测量链

## 2. 6DoF 与 5 连杆细分（含 L3/L4 详细尺寸）

6DoF 机械臂对应 5 个主连杆（不含底座）：

- `L1 = 180 mm`（基座安装面到 J2 轴心的垂直高度，底座立柱）

| 链节 | 连接关节 | 建议长度 | 截面建议 | 说明 |
|---|---|---:|---|---|
| Link-1 | J2 -> J3 | `L2 = 350 mm` | 7075 `70x45x3` 矩形管 | 大臂，主承载 |
| Link-2 | J3 -> J4 中置节点 | `L3a = 150 mm` | 7075 `60x40x3` | 前小臂前段 |
| Link-3 | J4 节点 -> J5 | `L3b = 100 mm` | 7075 `55x35x3` | 前小臂后段 |
| Link-4 | J5 -> J6 | `L4a = 55 mm` | 6061 `45x30x2.5` | 腕部偏转段 |
| Link-5 | J6 -> TCP | `L4b + Ltool = 35 + 100 mm` | 6061 `35x25x2` | 腕末端+工具段 |

总前伸半径（水平）：`R = 350 + 150 + 100 + 55 + 35 + 100 = 790 mm`  
垂直安装高度基线：`H = L1 = 180 mm`

### 2.1 中置偏转电机布置（重点）

- J4 偏转电机建议放在 `L3a/L3b` 分界处（距 J3 约 `150 mm`），通过短同步带驱动 J4 轴。
- 对比“J4电机放在腕部末端”，中置方案可把 J4 电机质量往近端移动，肩肘设计扭矩约下降 `1-2%`，同时减小远端惯量。
- 对 CATIA/SolidWorks 友好：中置节点只需要一个电机支架、一个驱动皮带轮座、一个被动轴承座。

## 3. 日字形截面是否更抗形变

### 3.1 结论先行

- 同外形尺寸、同壁厚时：日字形（双腔+中间腹板）抗弯更强，但明显增重。
- 同外形尺寸、同重量时：需要降壁厚，抗弯反而可能比普通矩形管略差。

### 3.2 定量结果（本项目截面）

| 截面 | 对比条件 | 惯性矩 I（强轴）变化 | 质量变化 |
|---|---|---:|---:|
| L2: 70x45x3 | 日字 vs 普通矩形管（同壁厚） | `+15.1%` | `+29.4%` |
| L3: 60x40x3 | 日字 vs 普通矩形管（同壁厚） | `+14.4%` | `+28.7%` |
| L2/L3 | 日字（等重降壁厚后）vs 普通矩形管 | 约 `-8.7%` | 约 `0%` |

设计建议：
- 大臂/小臂不建议全长都做日字形。
- 推荐“局部日字”：仅在根部高弯矩区（约前 `80-120 mm`）加中腹板，其余仍用普通矩形管。这样兼顾刚度、重量和建模复杂度。

## 5. 精度到 0.5mm 的提升路线（按影响优先）

## 5.1 当前误差主因

当前预算下，末端运动学误差贡献（理论）：

| 关节 | 当前输出角误差假设 | 对末端贡献 |
|---|---:|---:|
| J2 | `0.08 deg` | `1.10 mm` |
| J3 | `0.118 deg` | `0.91 mm` |
| J4 | `0.125 deg` | `0.42 mm` |
| J5 | `0.20 deg` | `0.35 mm` |
| J6 | `0.20 deg` | `0.18 mm` |

### 5.1.1 当前理论控制精度计算过程

定义：

- 单关节导致的末端线位移误差：`e_i = l_i * rad(DeltaTheta_i)`
- 控制链理论合成误差（RSS）：`e_ctrl = sqrt(sum(e_i^2))`

其中（按本版几何）：

- `l2 = L2 + L3a + L3b + L4a + L4b + Ltool = 0.79 m`
- `l3 = L3a + L3b + L4a + L4b + Ltool = 0.44 m`
- `l4 = L4a + L4b + Ltool = 0.19 m`
- `l5 = Ltool = 0.10 m`
- `l6 = 0.05 m`（末端滚转对 TCP 的等效半径）

代入当前假设角误差：

- `e2 = 0.79 * rad(0.08 deg) = 1.103 mm`
- `e3 = 0.44 * rad(0.118 deg) = 0.906 mm`
- `e4 = 0.19 * rad(0.125 deg) = 0.415 mm`
- `e5 = 0.10 * rad(0.20 deg) = 0.349 mm`
- `e6 = 0.05 * rad(0.20 deg) = 0.175 mm`

控制链理论精度（仅控制/传动角误差）：

- `e_ctrl = sqrt(1.103^2 + 0.906^2 + 0.415^2 + 0.349^2 + 0.175^2) = 1.537 mm`

若只把梁挠度（约 `0.08 mm`）并入，不含装配误差：

- `e_ctrl+deflection = sqrt(1.537^2 + 0.08^2) = 1.539 mm`

结论：当前理论控制精度约 `1.54 mm`，主导误差仍是 J2/J3。

## 5.2 达到 0.5mm 的必要角误差上限

若把运动学误差预算压到 `0.35 mm`（给结构与装配留余量），输出端角误差需约：

| 关节 | 目标误差上限 |
|---|---:|
| J2 | `<=0.011 deg` |
| J3 | `<=0.014 deg` |
| J4 | `<=0.016 deg` |
| J5 | `<=0.014 deg` |
| J6 | `<=0.012 deg` |

这比当前低成本舵机+普通同步带方案严苛很多。

## 5.3 优先改造顺序（兼顾力矩）

1. 先改 J2/J3：在输出轴新增绝对编码器（双环控制：电机编码器 + 关节编码器），并把减速比提升到 `J2: 3.2:1`, `J3: 2.2:1`。  
2. 再改 J4/J5：采用预紧分体皮带轮或零背隙齿轮副，先抑制回差，再做模型补偿。  
3. 结构与装配：J2/J3 轴系改双轴承预紧，法兰和轴承座增加定位止口，控制装配同轴度。  
4. 控制层：重力补偿 + 摩擦补偿 + 低速死区补偿 + S 曲线轨迹。  

关键判断：
- 只做“简单皮带减速”通常很难稳定到 `0.5 mm`。
- 要长期稳定 `0.5 mm`，J2/J3 需进入“高精关节级反馈 + 低回差传动”范式。

## 6. 机械减速方案（尽量简单，便于 CAD）

## 6.1 统一方案：单级同步带 + 槽孔张紧

| 关节 | 皮带规格 | 主/从轮齿数 | 比例 | 推荐中心距 |
|---|---|---|---:|---:|
| J2 | HTD-5M-15 | 20T / 50T | 2.5:1 | 105-115 mm |
| J3 | HTD-5M-15 | 20T / 34T | 1.7:1 | 80-90 mm |
| J4 | GT3-9 | 18T / 29T | 1.6:1 | 65-75 mm |
| J6 | GT2-6 | 16T / 24T | 1.5:1 | 32-40 mm |

机械建模最简原则：
- 每个减速级只保留 1 个主动轮、1 个从动轮、1 个张紧结构。
- 张紧优先用“电机底板长圆孔”（不先加复杂张紧器）。
- 轴系全部标准件：`双轴承 + 一体轴肩 + 止退螺母`。
- 需要中置驱动的 J4 仅增加 1 根短轴和 1 套支架。

## 6.2 建模顺序（CATIA/SolidWorks）

1. 先建关节中心骨架（J2/J3/J4/J5/J6 五个轴线）。  
2. 再按中心距布置皮带轮。  
3. 再生成支架和轴承座。  
4. 最后按干涉检查调整支架厚度和电机安装孔。  

## 7. 断电阻尼力矩（Power-off Damping / Backdrive）

## 7.1 官方公开情况

当前选型中，OCS 与 FEETECH 产品页通常给出额定扭矩、堵转扭矩、速度、电压、尺寸和重量，但多数未公开“断电阻尼力矩/反拖力矩”的明确数值。

因此：
- 官方可直接引用值：`未公开/无统一标准`
- 设计安全假设：`断电保持力矩按 0 N.m 处理`（不依赖断电自锁）

## 7.2 工程估算值（推断，非官方）

> 推断方法：按额定扭矩的 `3%-8%` 估计齿轮与轴承静摩擦形成的反拖阻尼。  
> 这些值仅用于仿真初值，最终必须台架实测。

| 型号 | 额定扭矩 (N.m) | 估算断电阻尼力矩范围 (N.m) | 备注 |
|---|---:|---:|---|
| OCS-D9501 | 34.32 | 1.03 - 2.75 | 推断值 |
| OCS-D6501 | 24.52 | 0.74 - 1.96 | 推断值 |
| OCS-D2002 | 8.34 | 0.25 - 0.67 | 推断值 |
| SM-260B-C001 | 6.37 | 0.19 - 0.51 | 推断值 |
| STS3250 | 1.57 | 0.05 - 0.13 | 推断值 |
| STS3215 | 0.98 | 0.03 - 0.08 | 推断值 |

## 7.3 实测方法（建议）

- 在输出轴安装已知力臂 `r`（例如 `0.10 m`）。  
- 缓慢施加拉力 `F`，记录“刚开始转动”的临界值。  
- 断电阻尼力矩：`T_d = F * r`。  
- 每个关节正反向各测 5 次，取均值与区间。
